Le Javascript :

Javascript est un language de programation de script employé pour rendre une page web interactive, le comportement de la page web et son apparence pourra varier en fonction de certain paramètres

Le PHP :

Le PHP est un language de programmation libre. utilisé pour générer des pages web dynamiques. qui sont donc modifier en fonction de certain information(heure, utilisateur,nom de l'utilisateur,ect. )

MySQL
